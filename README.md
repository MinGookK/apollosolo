#1 Set up

1. Create-React-App을 사용한다.
2. styled-component : CSS 위해 사용
3. react-router-dom : SPA 구현 위해 사용. 진짜 페이지 연결 외에 캐싱은 Apollo가 해준다.
4. Apollo + GraphQL : Apollo를 사용하면 GraphQL과 연동이 쉬워진다. 캐싱과 관련된 부분도 알아서 처리해준다.
